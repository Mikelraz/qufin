# qufin

A personal toolkit for quantitative analysis and modelling of financial market products, with a focus on performance. Built to support the development, training, and testing of trading strategies and models.

## Scope

- Pricing and analysis tools for financial instruments (equities, derivatives, fixed income, etc.)
- Data pipelines for market data ingestion and feature engineering
- Strategy backtesting and performance evaluation
- Statistical and machine learning models for signal generation
- Research notes as Jupyter notebooks

## Structure

```
qufin/
├── src/            # Core library code
├── notebooks/      # Research notes and exploratory analysis
├── tests/          # Unit and integration tests
├── data/           # Local data (not tracked by git)
└── docs/           # Reference documentation
```

## Status

Early stage — structure and scope evolve as topics are explored.
